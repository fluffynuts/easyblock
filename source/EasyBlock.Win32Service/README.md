# EasyBlock 

## Once-off run

`EasyBlock.Win32Service.exe -r`

## Installation instructions

1. Unpack this archive somewhere you'd like to keep the application
2. run `EasyBlock.Win32Service.exe --install`
3. run `net start easyblock`
